# Aplikasi Sistem Pemungutan Suara Digital (E-Voting)

Sistem backend E-Voting yang dibangun dengan Golang untuk mengelola data pemilihan dan menghitung perolehan suara secara cepat dan aman.

📋 Deskripsi Proyek
E-Voting adalah aplikasi untuk mengelola data pemilihan dan menghitung perolehan suara secara cepat. Data utama yang digunakan adalah data kandidat, data pemilih, dan jumlah perolehan suara. Pengguna aplikasi adalah panitia pemilihan dan anggota organisasi.

✨ Fitur Utama
1. Manajemen Data Kandidat
✅ Menambahkan data profil kandidat
✅ Mengubah data profil kandidat
✅ Menghapus data profil kandidat

2. Manajemen Data Pemilih
✅ Mencatat nomor urut, visi-misi, dan akumulasi poin suara masuk

3. Pencarian dan Pengurutan Data Kandidat
✅ Pencarian berdasarkan nomor urut menggunakan Sequential Search
✅ Pencarian berdasarkan nomor urut menggunakan Binary Search

4. Pengurutan Data Kandidat
✅ Pengurutan berdasarkan perolehan suara terbanyak atau nomor urut
✅ Menggunakan Selection Sort dan Insertion Sort

5. Statistik dan Laporan
✅ Menampilkan statistik persentase suara masing-masing kandidat dan total suara yang sudah masuk
